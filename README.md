# Muesum Analytic App

### Requirements
Node - [Download](https://nodejs.org/en/download/)

### Scripts 

To run locally

Install packages
```
npm i

```

Run start command

```
npm start

```
Development server will start at http://localhost:3000

<br/>
To run test

```
npm test

```
